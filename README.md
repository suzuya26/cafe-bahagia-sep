# CAFE BAHAGIA web
## Instalasi
1. Clone repo, cd to repo directory
2. set .env credentials
3. on terminal run :
- `composer update`
- `npm run dev`
4. open new terminal, do migration for db
`php artisan migrate`
5. to start dev server
`php artisan serve`
6. dont close `npm run dev` terminal, both should run at the same time while development
